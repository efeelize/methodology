# 1.规划：先确定总的目标和方法

> 在开始做一件毫无经验、不知从何入手的事情时，即使时间紧迫、压力巨大，也要自始至终保持着（甚至专门空出时间来练习）一种“全局观念”。对于眼前任务系统方法论的掌握、目的和实现路径的思索，永远都有利无害。相反，在没想清楚“为什么”和“怎么样”之前，贸然将法学院第一年最重要的资本——时间和精力投入进去，反而会有走错路、浪费宝贵机会成本的风险。

## 1.1 学习方案的制定

> 厘清了背后的规律后，我们会发现，变成一个高效自律的人并没有什么神奇魔法，而是需要一点一点脚踏实地的积累和改变。想要完成这个改变，又要求我们利用智慧和对自己的了解，为自己设计好训练频率、进阶路径，而不是许下一个过于宏伟的愿望，并发誓用意志力去坚持实现。

- 自己布置任务的时候，有几点需要注意:任务要符合自己的实际能力和时间安排，立下远大的志向并没有什么意义，但逐步完成目标会有真正的长期价值，不管它们看起来是多么微不足道。
- 任务的意义不一定是在于达成某种效果，比如，把某项成绩提高到多少分，而是在于坚持本身。每天大脑在把自己从其他事情切换出来、进入完成任务模式的时候，实际上就是在运用脑前额叶规范自己的行为，起到了锻炼自律和自控能力的效果。
- 训练自律的小任务一定要按照自己设定好的频率坚持完成，不能有今天请假明天补上的想法。因为这个过程最大的意义在于让脑前额叶去练习规范自我的能力，所以频率要比效果更重要。如果有一天出了意外情况，那可以减少任务的定量，比如，从健身一个小时改成健身半个小时，但应该坚持让脑前额叶完成一次自我规范的训练。
- 任务完成后，一次自律自控的训练周期就算完成。如果按照设定好的频率完成了所有任务，就应该给自己奖励，比如，做一些一直很想做的事情，再依此类推开启下一次训练周期。

### 1) 训练过程：先提高强度，再提高速度

- 对于考LSAT而言，拿几篇阅读题来读一下，看看自己能否像上文中的例子一样，自己理顺逻辑，形成框架。然后拿着自己写好的框架再去看题目，如果整体正确率能有90%以上的话，那准备LSAT的强度就达到了，剩下就是通过不断的模考来训练速度。
- 强度提高了之后，就要通过不断的训练来提高速度。速度的意思就在于，在同样的时间里，你是否能比别人抽象出更多层逻辑，对论证的思路和本质理解得更透彻，甚至能迅速找到文章/试题逻辑的薄弱点。
